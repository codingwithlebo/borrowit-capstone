# Borrowit — Founder Capstone

A neighbourhood tool and equipment lending marketplace. People near you have drills, mixers, tents, and bikes sitting idle. You need them for a day. Borrowit connects you.

## Setup

```bash
npm install
npm run dev
```

Type-check anytime:
```bash
npm run typecheck
```

---

## What's built

- **Browse screen** — search by keyword, filter by category and pricing
- **Item detail** — photo, description, owner info, ratings, Book Now
- **Booking flow** — 2 steps: pick dates (with live cost) → contact details → confirmation
- **Auth screen** — optional sign in / sign up (not forced — see FOUNDER-RESPONSE.md)

## What was deliberately cut

Messaging, map, wishlist, referral codes, dark mode, real-time updates, offline support. All documented with reasoning in `DECISION-LOG.md` and `FOUNDER-RESPONSE.md`.
